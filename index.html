Workflow

  1. โทรแจ้งหรือแจ้งผู้ใช้ให้เปิด TeamViewer QuickSupport
  2. ให้ผู้ใช้ส่ง Your ID และ Password ทางช่องทางที่ปลอดภัย
  3. เชื่อมต่อ -> ตรวจสอบปัญหา -> อธิบายขั้นตอนการแก่ -> ส่งไฟล์/อัปเดตดามต้องการ -> บันทึก session (กรณีถ้าจาเป็น)

เริ่มเชื่อมต่อ (Run remote)

  1. เปิด TeamViewer ทั้งสองเครื่อง
  2. ที่เครื่องถูกรีโมท ดู Your ID และ Password (6- or 8-digit)
  3. ที่เครื่องรีโมทเข้าไปที่ช่อง Partner ID → กด Connect
  4. ใส่รหัสผ่านที่ได้รับจากฝังถูกรีโมท → คลิก Log On
  5. ดอนเชื่อมจะมีแถบเครื่องมือสำหรับควบคุม, ส่งไฟล์, แชท, ปรับขนาดหน้าจอ

วิธีโอนไฟล์

  1. ใช้ฟีเจอร์ File Transfer ในแถบเครื่องมือ: เลือกโหมด Full remote หรือ File transfer แบบแยกหน้าต่าง
  2. สามารถลาก-วางไฟล์ระหว่างหน้าต่าง remote และ local ได้ (ขึ้น กับระบบปฏิบัติการ)

ปัญหาที่พบบ่อย & วิธีแก้

  1. ไม่เชื่อมต่อ (Connection failed): ตรวจสอบอินเทอร์เน็ต, firewall,proxy: ลอง restart โปรแกรม
  2. พบข้อความ “Access denied”: ดรวจสิทธิ์บัญชี, password ถูกต้อง หรือไม่, whitelist/blacklist
  3. หน้าจอไม่ชัด / หน่วง: ปรับคุณภาพการแสดงผลเป็น “Optimize speed” ในเมนู View → Quality
  4. ไม่เห็น Remote screen (black screen): บน macOS ต้องอนุญาต Screen Recording ใน System Preferences → Security & Privacy
  5. Session ถูกตัดบ่อย: ตรวจสอบความเสถียรอินเทอร์เน็ต ทั้งสองฝั่ง
